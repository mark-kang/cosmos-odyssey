# 🚀 Cosmos Odyssey (코스모스 오디세이)

웅장한 정통 우주 함대 전투를 모티브로 한 **동시 턴제(We-Go Turn-based) 전술 시뮬레이션 게임**입니다. 
양 플레이어가 동시에 이동 추진력과 선회각, 무기 타겟을 조작한 뒤 제출하면, 서버가 5.0초 동안의 물리 궤적과 피격 이벤트를 연동 계산하여 클라이언트에 동일하게 재생해 줍니다.

---

## 🎮 주요 게임 피처

* **우주 관성 물리 이동:** 함선은 우주 관성의 영향을 받습니다. 이전 턴에 이동하던 속도와 방향이 보존되며, 이번 턴에 입력한 추진력이 가산되어 최종 궤적이 도출됩니다.
* **선회각 제한:** 함선은 회전할 수 있는 최대 각도(60도 부채꼴 영역) 제약이 있어 정교한 기동 조타를 요구합니다.
* **전투 구역 화면 이탈 패널티 (Out-of-Bounds Damage):**
  * 캔버스 화면 바깥으로 나가는 관성 기동은 가능하나, 화면 밖으로 나갈 시 실시간 OOB 경고 오버레이가 작동합니다.
  * 서버 시뮬레이션 상 1.0초(10틱) 연속 이탈 시마다 `HP 10`씩 지속 대미지(DoT)를 입습니다.
  * 함선 분실을 막기 위해 화면 밖으로 나간 함선의 시각적 좌표는 화면 가장자리(20px 마진)에 안전하게 클램핑(고정)되어 재조작이 원활합니다.
* **무기 체계 2종:**
  * **직사 빔 (BEAM):** 2.5초 시점에 격발되며 함선에서 조준선까지 궤적 상에 적 함선이 걸치면 즉시 `25` 피해를 입힙니다. (키보드 단축키 `1`)
  * **범위 어뢰 (TORPEDO):** 0초에 격발하여 비행 속도(150px/s)로 조준점까지 날아가며, 폭발 지점 기준 60px 범위 내에 적 함선이 들어오면 거리에 따라 최대 `40` 데미지를 입힙니다. (키보드 단축키 `2`)
* **친절한 조작 가이드 UI:** 
  * 마우스 드래그를 올리고 내림에 따라 함선의 추진력 가이드선 및 예측 도착 고스트(Ghost), 선회각 부채꼴(각도기)이 실시간으로 갱신됩니다.
  * HUD 중앙에서 현재 선택 중인 무기를 가시성 있게 표시하며, 마우스 클릭(좌우 화살표) 및 단축키로 변경이 가능합니다.

## 🛠️ 프로젝트 구조 및 실행 가이드

프로젝트는 **Server(Node.js + Express)** 와 **Client(TypeScript + PixiJS + Vite)** 구조로 나뉘어 설계되었습니다.

### 1. Backend Server 실행
서버는 게임 룸 상태를 관리하고 5.0초간의 수학적 물리/피격 연산만 결정론적(Deterministic)으로 계산하여 클라이언트에 JSON 타임라인 데이터를 내려주는 API 역할을 합니다.
```bash
# server 디렉토리로 이동 및 패키지 설치
cd server
npm install

# 개발 모드로 서버 실행 (기본 포트: 3000)
npm run dev
```

### 2. Frontend Client 실행
클라이언트는 서버에서 응답받은 JSON 데이터를 50틱 타임라인에 따라 부드럽게 이동, 사격, 피격 연출을 수행하는 시퀀서 뷰어 역할을 합니다.
```bash
# client 디렉토리로 이동 및 패키지 설치
cd client
npm install

# 로컬 개발 서버 실행 (기본 포트: 5173)
npm run dev
```

## 🕹️ 조작법 가이드

1. **함선 선택 및 이동 조작:**
   * 자신의 함선 본체(빛나는 전함)를 **마우스 좌클릭 후 드래그**하여 추진력 벡터 크기와 방향을 조절합니다.
2. **무기 선택 및 발사 좌표 설정:**
   * 키보드 단축키 **`1`** 또는 HUD 무기창의 **`>`** 버튼을 클릭해 **직사 빔 (BEAM)** 무기를 조착할 수 있습니다.
   * 키보드 단축키 **`2`** 또는 HUD 무기창의 **`<`** 버튼을 클릭해 **범위 어뢰 (TORPEDO)** 무기를 장착할 수 있습니다.
   * 화면 캔버스 상에서 **마우스 우클릭**을 하면 조준점(크로스헤어)이 지정되며 함선과 이어집니다.
3. **명령 제출:**
   * 조작이 마음에 들면 하단의 **명령 제출** 버튼을 누르거나 **`Space`** 키를 눌러 턴 명령을 서버로 보냅니다.
   * 두 플레이어 모두 제출이 완료되면 시뮬레이션(5초)이 자동 재생됩니다.
