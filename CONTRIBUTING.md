## How to Contribute

1. **Fork the Repository:** Create your own fork of this repository.
2. **Create a Branch:** Create a branch for your feature or bug fix (e.g., `git checkout -b feature/new-weapon`).
3. **Write Code & Verify:** Ensure your code passes TypeScript compilation (`npm run build`) and contains no errors.
4. **Submit a Pull Request (PR):** Submit a PR against the `main` branch.
5. **Reference the CLA:** In your PR description, please include the statement: *"I agree to the Contributing Guidelines and the Contributor License Agreement (CLA) of this project."*
